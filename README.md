#  Europe Electricity Load Forecasting

This project focuses on analyzing European electricity load data and developing predictive models for future demand.

---

## 📂 Project Structure

* **data/**: Raw and processed datasets (.csv)
* **notebooks/**: All coding work including cleaning, analysis, and modeling (.ipynb)
* **output/**: Visualizations, plots, and model results
* **docs/**: Project reports, presentations, and documentation
* **scripts/**: Reusable Python helper functions (.py)

## 🚀 Working Rules

1. **Isolation:** Each group must create their own notebook in the `notebooks/` folder (e.g., `1_data_cleaning.ipynb`).
2. **Sync First:** Always `PULL` the latest changes before you start working and before you `PUSH` your code.
3. **Code Quality:** Do not push broken or non-functional code to the main branch.
